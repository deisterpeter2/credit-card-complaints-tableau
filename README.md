# credit-card-complaints-tableau
This project analyzes consumer credit card complaints using Tableau to identify complaint trends, response performance, and common customer issues.

The interactive dashboard enables users to explore complaint volume over time, identify high-volume complaint categories, and monitor key customer service performance metrics.

https://public.tableau.com/views/Credit_Card_Complaint_Project/Dashboard1?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link


Objectives
-- Analyze trends in customer complaints
-- Identify the most common complaint categories
-- Measure complaint response performance
-- Build an interactive dashboard for business users
-- Demonstrate Tableau dashboard design and data visualization skills

Dashboard Features
KPI Summary
-- Total Complaints
-- Timely Responses
-- Timely Response Rate (%)
-- In-Progress Complaints
-- In-Progress Complaint Rate (%)

Complaint Trend
-- Line chart displaying complaint volume over time
-- Interactive date range filter

Complaint Heat Map
-- Heat map highlighting complaint distribution
-- Quickly identifies areas with the highest complaint volume

Top 10 Complaint Categories
-- Ranked bar chart showing the most frequently reported complaint types
